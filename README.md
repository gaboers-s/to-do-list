# Dailist 📝

A simple **Todo List web app** built with **HTML, CSS, and JavaScript**.

This project is part of my practice in learning **frontend development and DOM manipulation**.

---

## Features

* Add tasks
* Mark tasks as completed
* Delete tasks
* Filter tasks by priority
* Task progress tracking

---

## HTML Structure

The project uses semantic HTML elements for better structure.

Main sections:

* **Header** – App title and description
* **Progress Bar** – Shows task completion progress
* **Add Task Form** – Input field and add button
* **Filter Buttons** – Filter tasks by priority
* **Task List** – Displays all tasks
* **Footer** – Task statistics

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
